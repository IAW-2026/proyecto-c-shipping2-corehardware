# shipping

Aplicación **Shipping** del [Proyecto IAW 2026](https://iaw-2026.github.io/proyecto/) — comisión Corehardware.

Esta app corresponde al módulo de envíos y logística en el proyecto de tipo **C (Marketplace)**.

---

**Dependencias**:
- **Framework**: Next.js
- **Autenticación**: Clerk
- **Estilos**: Tailwind
- **ORM**: Prisma
- **Base de datos**: PostgreSQL (Neon)
- **Deploy**: Vercel
